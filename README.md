# Lume Boutique

Sistema de loja virtual feminina desenvolvido em Python e MySQL.

## Tecnologias

- Python
- MySQL
- MySQL Connector

## Funcionalidades

- Cadastro de usuários
- Login
- Controle de estoque
- Carrinho de compras
- Finalização de pedidos
- Painel administrativo

## Banco de Dados

Importe o arquivo:

lume_boutique.sql

## Executar o projeto

Instalar dependências:

pip install -r requirements.txt

Executar:

python main.py
